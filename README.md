# Altaura Website

This project contains a multi-page marketing site for Altaura with editorial styling, shared navigation, and conversion-focused pages.

## Structure
- index.html — Home
- about.html — About
- services.html — Services
- portfolio.html — Portfolio
- ebooks.html — Resources
- contact.html — Contact
- css/styles.css — Design system and page styles
- js/main.js — Nav, animation, filters, accordions, and ebook capture
